# Sales-Analytics

## Опис
У цьому проєкті виконано повний аналіз даних по продажам компанії, акаунтам та сесіям у розрізі країн, континентів, девайсів, каналів. А також проведено аналіз кореляцій та порівняння різних груп клієнтів

## Отримання даних

За допомогою Python виконано підключення до Google BigQuery та сформовано фінальний датасет.

Було застосовано JOIN таблиць (LEFT JOIN, INNER JOIN) - кінцевий датасен сформований з 5ти різних таблиць таким чином, щоб зберегти всі сесії та замовлення, навіть для незареєстрованих користувачів.

## Опис набору даних

Датасет для подальшого аналізу має 18 колонок наступного типу:

1. 5 колонок числового типу, а саме:

- ідентифікатор сесії
- ідентифікатор акаунту
- чи підтвердив користувач свій email (якщо так, то 1, якщо ні то 0)
- чи підписався користувач на email розсилку (якщо так, то 0, якщо ні то 1)
- ціна товару

2. 12 колонок категоріального типу, а саме:

- континент
- країна
- девайс
- браузер
- назва моделі пристрою
- операційна система
- мова браузера
- інформація про джерело трафіку
- канал трафіку
- категорія товару
- назва товару
- короткий опис товару
  
 3. Одна колонка типу datetime, а саме - дата

 Додатково отримано:
 - кількість унікальних сесій 349545
 - період аналізу від 2020-11-01 до 2021-01-31
 - ми маємо користувачів з 6 континентів та 108 різних країн
 - компанія реалізує продукцію 14 різних категорій

## Аналіз даних та динаміка

Проведено аналіз даних та динаміки в розрізі різних показників та категорій, а саме:

- проаналізовано ТОП-3 континента та ТОП-5 країн за продажами та замовленнями
- проаналізовано ТОП-10 категорій за доходом та ТОП-10 категорій у ТОП-5 країнах
- проведено аналіз продажів за девасами та основними моделями, які використовують покупці
- досліджено розподіл продажів за джерелами трафіку
- проведено аналіз користувачів, які підтвердили свою електронну адресу та які відписались від розсилки загалом та по континентам
- проаналізовано кількість зареєстрованих акаунтів по країнам
- проведено аналіз динаміки продажів в рохрізі днів та тижнів
- досліджено динаміку продажів в розрізі ТОП-3 континента, ТОП-5 категорій
- на континенті з найбільшими продажами досліджено динаміку ТОП-5 категорій та ТОП-5 країн
- проаналізовано динаміку продажів по каналам та девайсам

Створено зведені таблиці:

- сесії за каналами та девайсами
- продажі за категоріями та країнами
- продажі серед зареєстрованих та незареєстрованих користувачів
- розподіл кількості сесій та сум продажів між мовами та типами девайсів

## Аналіз кореляцій та статистичні тести

Досліджено взаємозв’язки між показниками:

- сесії та продажі
- продажі по континентах
- продажі по каналам
- продажі по категоріях
- продажі по мовам які використовують покупці

Розраховано:
- коефіцієнт кореляції
- статистичну значущість

Проведено статистичні тести для:
- порівняння зареєстрованих vs незареєстрованих користувачів
- аналіз відмінностей між каналами трафіку
- аналіз частки органічного трафіку
- аналіз відмінностей між продажами та кількістю сесій в Америці та на інших континентах

Використано:
- t-test
- тест Mann-Whitney
- тест Shapiro-Wilk
- z-test
- тест Kruskal - Wallis
- тест Spearmanr
